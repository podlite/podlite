## build

Generate JavaScript documentation as a list of parsed JSDoc
comments, given a root file as a path.
~~~~~~~
{
  "type": "block",
  "name": "pod",
  "content": [
    {
      "type": "block",
      "level": 2,
      "name": "head",
      "content": [
        "build"
      ]
    },
    {
      "type": "para",
      "content": [
        "Generate JavaScript documentation as a list of parsed JSDoc\ncomments, given a root file as a path."
      ]
    }
  ]
}
